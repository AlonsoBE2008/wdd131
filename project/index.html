About this project

This site my student project for WDD131. The goal is to demonstrate responsive HTML, accessible markup, CSS design, and JavaScript interactivity. Content is original and written for this project.

  • Target audience: Home cooks, beginners, busy students.
  • Design goals: Readability, fast load, mobile-first layout.

What you'll find

  • Clear recipes with ingredient lists and step-by-step instructions.
  • Optimized images with lazy loading for faster pages.
  • A contact form that saves messages to localStorage for demonstration.
